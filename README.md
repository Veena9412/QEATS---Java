# QEATS---Java
QEATS is a Java-based food ordering application built as part of the Crio Buildout series. It demonstrates REST API development, database integration, and modular design for managing restaurants and orders. The project helps learners apply OOP principles, exception handling, and test-driven development in a real-world context.
Features:
1. RESTful APIs for restaurant search and food ordering
2. CRUD operations for restaurants and orders
3. Database persistence using JPA/Hibernate
4. Input validation and error handling
5. Unit testing with JUnit
